# 快速开始指南

## 🚀 5分钟快速上手

### 第一步：打开项目

1. 使用微信开发者工具打开项目
2. 配置 AppID（在项目设置中）

### 第二步：开通云开发

1. 点击微信开发者工具顶部的"云开发"按钮
2. 开通云开发服务（首次需要）
3. 创建环境（选择"基础版"即可）
4. 记录环境 ID

### 第三步：创建数据库集合

在云开发控制台 → 数据库 → 创建集合：

**集合1：training_records**
- 权限：所有用户可读，仅创建者可写

**集合2：subscribe_settings**
- 权限：仅创建者可读写

### 第四步：运行项目

1. 点击"编译"按钮
2. 在模拟器中查看效果
3. 首次进入会显示"安全须知页"

### 第五步：测试流程

1. ✅ 阅读安全须知，勾选同意
2. ✅ 进入首页，选择训练时长和呼吸节奏
3. ✅ 点击"开始"按钮，进入训练页
4. ✅ 观察呼吸引导动画和进度环
5. ✅ 完成训练后查看完成页
6. ✅ 查看历史记录页

## 📝 重要配置说明

### 1. 云开发环境 ID（可选）

如果需要指定环境，在 `miniprogram/app.js` 中修改：

```javascript
wx.cloud.init({
  env: 'your-env-id', // 替换为你的环境ID
  traceUser: true,
})
```

### 2. 订阅消息模板 ID（可选）

在 `miniprogram/pages/subscribe/index.js` 中替换：

```javascript
wx.requestSubscribeMessage({
  tmplIds: ['your-template-id-1', 'your-template-id-2'],
  // ...
})
```

**获取模板 ID**：
1. 小程序后台 → 功能 → 订阅消息
2. 申请模板
3. 获取模板 ID

### 3. 图标资源（可选）

如果图标不显示，可以在 `miniprogram/images/icons/` 目录下添加图标文件，或使用 iconfont。

## ⚠️ 常见问题

### 问题1：页面空白

**解决**：检查 `app.json` 中的页面路径是否正确

### 问题2：云开发报错

**解决**：确保已开通云开发并创建环境

### 问题3：Canvas 不显示

**解决**：确保基础库版本 ≥ 2.9.0

### 问题4：数据保存失败

**解决**：检查数据库集合权限设置
